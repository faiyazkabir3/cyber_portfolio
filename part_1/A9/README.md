# A9 – Privacy Technique Used Online

## Activity Description
This activity investigates privacy techniques used online to protect user information while interacting with internet services. The goal is to identify a commonly used privacy protection method and explain how it helps protect personal data.

---

## Definition: Online Privacy Techniques

Online privacy techniques are methods used to protect personal information when users browse the internet or interact with online services. These techniques help prevent unauthorized tracking, monitoring, or interception of sensitive information such as browsing activity, location, and login credentials (Whitman & Mattord, 2022).

---

## Discovery: Virtual Private Network (VPN)

A widely used online privacy technique is the **Virtual Private Network (VPN)**.

A VPN protects internet privacy by encrypting a user’s internet traffic and routing it through a secure server before reaching its destination.

VPN services are commonly used to:

- Protect data on public Wi-Fi networks
- Hide the user’s real IP address
- Prevent third parties from monitoring browsing activity

---

## How VPN Works

When a user connects to a VPN:

1. The device establishes a secure encrypted connection with the VPN server.
2. All internet traffic is routed through this encrypted tunnel.
3. The VPN server forwards the traffic to the destination website.

This process hides the user’s real IP address and encrypts transmitted data. Encryption protocols such as **OpenVPN** or **WireGuard** ensure that intercepted traffic cannot easily be read by attackers (NIST, 2020).
