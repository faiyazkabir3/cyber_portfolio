# A11 – Access Control Devices

## Activity Description
This activity investigates devices used to control access to secure areas or systems. The goal is to identify different access control technologies used in physical and digital security environments.

---

## Definition: Access Control Devices

Access control devices are physical or electronic systems used to regulate who can enter or use a restricted area, system, or resource. These devices authenticate users before granting access to secure locations or systems (Stallings & Brown, 2018).

---

## Discovery: Examples of Access Control Devices

Five examples of access control devices include:

1. **Biometric fingerprint scanners** – authenticate users using unique fingerprint patterns.
2. **Facial recognition access systems** – use facial recognition technology to verify identity.
3. **RFID access cards** – allow entry to buildings using radio-frequency identification cards.
4. **Smart locks** – electronic locks controlled using mobile applications or digital credentials.
5. **Keypad entry systems** – require a PIN code before granting access.

These devices are commonly used in offices, universities, government buildings, and residential security systems.
