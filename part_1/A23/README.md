# A23 – Enhancing Cybersecurity at Home

## Definition

Home cybersecurity refers to practices used to protect personal devices, networks, and data from cyber threats such as unauthorized access, malware, and data breaches (Whitman & Mattord, 2022).

---

## Activity Description

In this activity, I implemented several cybersecurity improvements on my personal devices and home network to enhance security and privacy.

---

## Security Enhancements Implemented

### 1. Limited IP Tracking

Enabled privacy features to limit IP tracking on my Wi-Fi connection, reducing exposure of network activity.

---

### 2. Multi-Factor Authentication (MFA)

Enabled MFA using **Microsoft Authenticator**.

- Adds an extra layer of security  
- Prevents unauthorized access even if passwords are compromised  

---

### 3. System and App Updates

Updated all applications and operating systems on:

- iPhone  
- Mac  

This ensures security vulnerabilities are patched.

---

### 4. App Permission Restrictions

Removed unnecessary permissions from apps:

- Location access  
- Microphone access  

This improves privacy and reduces data exposure.

---

### 5. FileVault Encryption

Enabled **FileVault** on Mac.

- Encrypts all stored data  
- Protects data if the device is lost or stolen
