# A12 – Offline Security Tools

## Activity Description
This activity investigates security tools that operate without requiring an active internet connection. These tools help protect systems, detect threats, and secure sensitive information locally.

---

## Definition: Offline Security Tools

Offline security tools are software or hardware tools used to protect computer systems and data without relying on an active internet connection. These tools help detect threats, secure data, and protect systems from unauthorized access (Whitman & Mattord, 2022).

---

## Discovery: Examples of Offline Security Tools

Five examples of offline security tools include:

1. **Antivirus software** – scans systems for malware and removes malicious files. example: MS Defender
2. **Disk encryption tools** – encrypt stored data to prevent unauthorized access. example: BitLocker
3. **Password managers** – securely store and manage strong passwords. example: KeePassXC
4. **Backup software** – creates offline backups of important files. example: Windows File History
5. **System recovery tools** – restore systems after security incidents or failures. example: Windows Recovery Environment

These tools are commonly used to maintain system security even when devices are disconnected from the internet.
