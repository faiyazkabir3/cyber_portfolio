# A10 – Privacy Technique Used Offline

## Activity Description
This activity investigates privacy techniques used offline to protect sensitive information stored on devices or storage media.

---

## Definition: Offline Privacy Techniques

Offline privacy techniques refer to methods used to protect personal information stored on devices or physical storage systems. These techniques prevent unauthorized individuals from accessing sensitive data stored locally on computers, smartphones, or storage devices (Whitman & Mattord, 2022).

---

## Discovery: Full-Disk Encryption

A common offline privacy technique is **full-disk encryption**, which protects stored files using cryptographic algorithms such as the **Advanced Encryption Standard (AES)**.

---

## How Full-Disk Encryption Works

Full-disk encryption encrypts all data stored on a storage device using a symmetric encryption algorithm.

When encryption is enabled:

1. Files are automatically encrypted when written to disk.
2. Data is decrypted only when the user successfully authenticates.
3. Authentication may require a password, PIN, or biometric verification.

Without the correct authentication credentials, encrypted data remains unreadable (NIST, 2001).
