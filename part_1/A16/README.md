# A16 – Local Security Incidents

## Activity Description
This activity investigates cybersecurity incidents affecting organizations or infrastructure within Australia using official government cybersecurity alerts.

---

## Definition: Local Security Incidents

Local security incidents refer to cybersecurity threats, vulnerabilities, or attacks that affect organizations, infrastructure, or systems within a specific country or region. Monitoring national cybersecurity alerts helps organizations identify risks and apply security updates to prevent exploitation (Australian Cyber Security Centre, 2026).

---

## Discovery

1. **Exploitation of Cisco SD-WAN appliances (2026)** – The Australian Cyber Security Centre reported active exploitation targeting Cisco Software-Defined Wide Area Network (SD-WAN) devices. These vulnerabilities may allow attackers to gain unauthorized access to network infrastructure if systems remain unpatched (Australian Cyber Security Centre, 2026).

2. **Remote code execution vulnerability in n8n workflow automation platform (2026)** – A critical vulnerability (CVE-2026-21858) was discovered that allows attackers to execute malicious code on affected systems, potentially leading to unauthorized system access (Australian Cyber Security Centre, 2026).

3. **MongoDB server vulnerability exploitation (2025)** – The Australian Cyber Security Centre warned about vulnerabilities affecting MongoDB servers that could expose sensitive data if security patches are not applied (Australian Cyber Security Centre, 2026).
