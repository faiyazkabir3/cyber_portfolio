# A13 – Online Security Tools

## Activity Description
This activity investigates online security tools that help detect vulnerabilities, analyze threats, and protect systems from cyberattacks. These tools are widely used by cybersecurity professionals to monitor systems and identify potential security risks.

---

## Definition: Online Security Tools

Online security tools are software services that operate through the Internet to help detect vulnerabilities, analyze threats, and protect systems from cyberattacks. These tools are commonly used by cybersecurity professionals to monitor networks, scan files for malware, and identify exposed systems (Whitman & Mattord, 2022).

---

## Discovery

Five examples of online security tools include:

1. **VirusTotal** – an online malware analysis service that scans files and URLs using multiple antivirus engines to detect malicious software (Google, 2023).
2. **Shodan** – a search engine that identifies internet-connected devices and helps researchers discover exposed servers or vulnerable systems (Shodan, 2023).
3. **Have I Been Pwned** – a service that allows users to check whether their email address has appeared in known data breaches (Hunt, 2023).
4. **Qualys SSL Labs** – an online service that analyzes SSL/TLS configurations of websites and identifies security weaknesses (Qualys, 2023).
5. **Cloudflare Web Application Firewall (WAF)** – an online security service that protects websites from attacks such as SQL injection, cross-site scripting, and distributed denial-of-service attacks (Cloudflare, 2023).
