# A14 – AI-Enabled Security Solutions

## Activity Description
This activity investigates security technologies that use artificial intelligence to detect cyber threats, analyze security data, and automate threat detection processes.

---

## Definition: AI-Enabled Security Solutions

AI-enabled security solutions are cybersecurity systems that use artificial intelligence and machine learning to detect threats, analyze large amounts of security data, and automatically respond to cyberattacks. These technologies help identify abnormal patterns of behavior that may indicate malicious activity (Stallings & Brown, 2018).

---

## Discovery

Five examples of AI-enabled security solutions include:

1. **Darktrace** – an AI-powered cybersecurity platform that uses machine learning to detect abnormal network activity and potential cyber threats in real time (Darktrace, 2023).
2. **CrowdStrike Falcon** – an endpoint security platform that uses artificial intelligence to detect malware, ransomware, and advanced cyber threats (CrowdStrike, 2023).
3. **IBM QRadar** – a security information and event management (SIEM) platform that uses AI analytics to detect suspicious behavior across networks (IBM, 2023).
4. **Microsoft Defender for Endpoint** – uses machine learning models to identify malicious activity and advanced persistent threats on devices (Microsoft, 2023).
5. **Vectra AI** – an AI-driven network detection and response system that analyzes network traffic to identify hidden cyber threats (Vectra AI, 2023).
